* _Visual_Studios_Coder_
* _CSS_
* _HTML_
* _GIT_BASH_
* _Javascript_

## Description

_This program is a showcase of using radio buttons score a person for everychoice they select and then give them a suggested language to learn. Repo is located at [https://github.com/MatthewJbelcher7/P-L-S]

## Setup/Installation Requirements

* _Clone this repository to your desktop_
* _Open index.html in your browser_
* _Awnser a varity of question to better understand where you should begin as a programer_
* _Click submit after makign all selections and you should be promted with our selection
* _Press try again to clear the screen
